# TSRPC Server

## Run
### Local Dev Server
```
npm run dev
```



### Build
```
npm run build
```

---

## Files
### Generate ServiceProto
```
npm run proto
```

### Generate API templates
```
npm run api
```

### Sync shared code to client

```
npm run sync
```

> If you chose symlink when using `create-tsrpc-app`, it would re-create the symlink instead of copy files.
